# 📚 AI Study Companion

## 📖 Overview

AI Study Companion is an intelligent study assistant that helps students learn more effectively using Artificial Intelligence. Users can upload PDF study notes, ask questions, generate quizzes, and create flashcards.

## ✨ Features

- 📄 Upload PDF Notes
- 🤖 AI Question Answering
- 🧠 RAG (Retrieval-Augmented Generation)
- 🔍 FAISS Semantic Search
- 📝 Quiz Generator
- 🃏 Flashcard Generator
- 💬 Chat Interface
- 🎨 Neon UI

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini
- FAISS
- Sentence Transformers
- PyPDF

## 🚀 How to Run

1. Install requirements
2. Run:

```bash
streamlit run app.py
```
